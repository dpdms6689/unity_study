C# 자료형 정리

정수형 (Integral Types)

1. int
    - 크기: 32비트
    - 범위: -2,147,483,648 ~ 2,147,483,647
    - 용도: 기본 정수형으로 가장 많이 사용되며, 일반적인 숫자 계산에 적합
    - 예시: `int age = 30;`
2. long
    - 크기: 64비트
    - 범위: -9,223,372,036,854,775,808 ~ 9,223,372,036,854,775,807
    - 용도: 매우 큰 범위의 정수를 필요로 할 때 사용
    - 예시: `long distance = 9876543210L;`

실수형 (Floating-point Types)

1. float
    - 크기: 32비트
    - 정밀도: 약 7자리
    - 용도: 소수점을 포함하는 숫자, 메모리 사용을 줄이면서도 실수 계산이 필요한 경우
    - 예시: `float pi = 3.14f;`
2. double
    - 크기: 64비트
    - 정밀도: 약 15~16자리
    - 용도: 보다 높은 정밀도가 요구되는 실수 계산에 사용
    - 예시: `double e = 2.718281828459045;`

논리형 (Boolean Type)

1. bool
    - 크기: 1비트 (실제로는 내부적으로 최적화됨)
    - 값: `true` 또는 `false`
    - 용도: 조건식, 논리 연산, 제어 흐름 등에 사용
    - 예시: `bool isRaining = false;`

문자형 (Character Type)

1. char
    - 크기: 16비트
    - 용도: 단일 유니코드 문자 저장
    - 예시: `char initial = 'A';`

문자열형 (String Type)

1. string
    - 크기: 유니코드 문자들의 시퀀스 (동적 길이)
    - 용도: 텍스트 데이터 저장 및 처리
    - 예시: `string name = "Alice";`

객체형 (Object Type)

1. object
    - 크기: 모든 데이터 타입의 기본 클래스
    - 용도: 모든 데이터 타입을 담을 수 있는 일반적인 참조형 변수
    - 예시: `object obj = 42;`
